해당 문제는 사용자의 Git 구성이 올바르게 설정되지 않아 발생하는 것. 일반적으로 Git 커밋을 할 때, 사용자의 이름과 이메일이 설정되어 있어야 함. 이 정보는 커밋 메시지에 포함되기 때문에 필요함.

문제를 해결하기 위해 다음 단계:

Git의 사용자 이름과 이메일을 설정. 다음 명령어를 사용하여 Git 사용자 이름과 이메일을 설정.

bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

여기서 "Your Name"과 "your.email@example.com"을 실제 사용자 이름과 이메일 주소로 바꿔야 함.

이후에는 VSCode에서 커밋을 시도. 이제는 사용자 정보가 올바르게 설정되었기 때문에 커밋이 정상적으로 진행될 것.
